Ad and LLM Demo

Requires Go be installed and on the path. 

API Keys:
- OpenNode: Dev API key for testnet transactions required: https://app.dev.opennode.com/signup. Add API key to `main.go`
- OpenAI: Add OpenAI API key to `server.go`

To start the server, run `go run *.go` in the terminal. The ad demo will be available at http://localhost:8080 and the LLM demo will be available at http://localhost:8080/llm
